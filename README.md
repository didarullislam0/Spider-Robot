# Spider Robot 🕷️🤖

## Project Description
This **Spider Robot** is an 8-legged autonomous robot powered by Arduino and Servo Motors.

## Features
- 🕷️ 8 Servo Motors used.
- 🔋 Battery powered.
- 🧠 Controlled by Arduino.
- 🎮 Remote control support.

## Required Hardware
- Arduino Uno
- Servo Motor (8x)
- Ultrasonic Sensor
- Li-Po Battery
